# Spec Driven Development
For Spec Driven Development, i have reverse engineered the My Journal application code generated in vibe coding and generated the specs as "journal-app-analysis".

Once the specs are generated, I have added new feature "add-journal-search" to enhance the functionality. In both the cases I have used OpenSpec 1.2

## Command
Used the following to add "journal search" feature

*/opsx-propose  Add a new feature “journal search” to journal-app using journal name and display the list of journals matching the search criteria. By default it should display all journals when there is no search criteria. This page will be the default home page.*
